Dual Display，Ubuntu下为笔记本实现双显示器

发表于 2008-09-16   |   分类于 Linux App

ref ： http://thatssky.spaces.live.com/Blog/cns!251B8640E925FC00!507.entry

修改/etc/X11/xorg.conf：

Section "Device"
    Identifier    "Configured Video Device"
    Driver        "fglrx"
EndSection

# added
Section "Device"
    Identifier    "Card1"
    Driver        "fglrx"
EndSection

Section "Monitor"
    Identifier    "Configured Monitor"
EndSection

# added
Section "Monitor"
    Identifier    "ExtMonitor"
EndSection

Section "Screen"
    Identifier    "Default Screen"
    Monitor        "Configured Monitor"
    Device        "Configured Video Device"
    Defaultdepth    24
EndSection

# added
Section "Screen"
    Identifier    "Screen1"
    Monitor        "ExtMonitor"
    Device        "Card1"
    Defaultdepth    24
EndSection

#Section "ServerLayout"
#    Identifier    "Default Layout"
#        screen          "Default Screen"
#    Inputdevice    "Synaptics Touchpad"
#EndSection

# added
Section "ServerLayout"
    Identifier    "DualHead"
        screen          "Default Screen" 0 0
        screen          "Screen1" RightOf "Default Screen"
    Inputdevice    "Synaptics Touchpad"
EndSection

＃如果只是扩展屏幕，也就是窗口可以拖到另外一个屏幕，就加上下面这段；如果需要两个独立的显示就去掉。
# added
Section "ServerFlags"
        Option "Xinerama" "true"
EndSection

【zz】linux内核参数
upstart : Ubuntu's event-based init daemon
