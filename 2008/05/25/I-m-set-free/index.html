I'm set free

发表于 2008-05-25   |   分类于 Life

        突然有了一种释放的感觉，晚上围着校园走了一圈，很惊奇的发现居然觉得草丛里
的花朵很漂亮了，还可以闻到一股清新的气息。我总觉得类似这种自然景观的欣赏从来都是与我无关
的，可是现在仿佛能体会到为什么游山玩水时应该感到兴奋。想到一句话：内心的执着，是
会遮挡住眼前的风景的。以前也曾几次用疑惑的眼神问朋友，到底旅游有什么好玩的，没有找到
能充分说服自己的答案，今天回过头一看，我所找寻的不就是这句话。
       再次感受到“简单的道理”，可是再简单的道理，自己往往弄不明白，直到身边有人提醒，才
明白真正的答案是什么。也许这只不过一句玩笑，或者一句看似隔了十万八千里的无关词句。

       free应当是一种积极的态度，是受限制的free，就像“有规律的娱乐”。
       There are so many times that we think something is definitely impossible to happen, but it happened, and something that we are looking forward so much, but it failed to appear. That’s life!
       So live in reality. That’s also the very reason why I favoure tragedy, which always remind you that you are living in a real world.

       不再期待，一个充实的自己就是最好的期待，安排好自己的行程上路了。
