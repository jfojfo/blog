3个解析url的php函数

pathinfo

<?php
$test = pathinfo(“http://localhost/index.php“);
print_r($test);
?>
结果如下
Array
(
[dirname] => http://localhost //url的路径
[basename] => index.php //完整文件名
[extension] => php //文件名后缀
[filename] => index //文件名
)

 

parse_url

<?php
$test = parse_url(“http://localhost/index.php?name=tank&sex=1#top“);
print_r($test);
?>
结果如下
Array
(
[scheme] => http //使用什么协议
[host] => localhost //主机名
[path] => /index.php //路径
[query] => name=tank&sex=1 // 所传的参数
[fragment] => top //后面根的锚点
)

Parse query string into an array:

$str = ‘pg_id=1&parent_id=2&document=doc&video’;
parse_str($str, $arr);
print_r($arr);

===>Array ( [pg_id] => 1 [parent_id] => 2 [document] => doc [video] => )

或者 $_GET[“pg_id”] ==> 1

$str = “first=value&arr[]=foo+bar&arr[]=baz”;
parse_str($str);
echo $first; // value
echo $arr[0]; // foo bar
echo $arr[1]; // baz

 

basename

<?php
$test = basename(“http://localhost/index.php?name=tank&sex=1#top“);
echo $test;
?>
结果如下
index.php?name=tank&sex=1#top
